#Read R, G, IR reflection light to the CSV file

##Operation
Step 1: Open the main program
index.html

Step 2: Connect to the device
Click 'Connect to device' button

Step 3: Record
click 'start record' button

##Format
time, green-light value, red-light value, IR-light(940nm) value, packet number
