# Streaming auido data from node server to browser
## (a) MP3 version
Step 1: Start the server to deliver the audio stream
node server_mp3.js

Step 2: Client: Browser to listen the audio
[browser] http://localhost:8001/socket_mp3.html

## (b) WAV version
Step 1: Start the server to deliver the audio stream
node server_mp3.js

Step 2: Client: Browser to listen the audio
[browser] http://localhost:8001/socket_mp3.html
