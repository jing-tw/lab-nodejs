# Install
```bash
yarn install
```

# Web app build
``` bash
yarn build
```

# Web app test
## start the webpack dev server
```bash
yarn start:dev
```

## test
```bash
# Open another terminal
yarn build
yarn test
```

# Electron app build and run
```bash
yarn electron-app:all
```
